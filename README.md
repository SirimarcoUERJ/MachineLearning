## 🚀 Como Executar
  1. Clone o repositório:
     ```
     git clone https://github.com/SirimarcoUERJ/MachineLearning.git
     cd MachineLearning
     ```
  2. Instale as dependências:

      ```
      pip install -r requirements.tx
      ```

  3. Execute os notebooks:

      ```
      jupyter notebook
      ```
